Naming Style 命名风格

  • Class/Struct 类/结构体
    • "名词"或"形容词+名词",驼峰式. 比如 Table, GreatTable.
  • Constant/Macro 常量/宏
    • "名词"或"形容词+名词",全大写 ,比如 ARRAYLENGTH.
  • Method/function 方法/函数
    • "动词" 或 "动词+名词" ,动词小写,名词大写. 如 getName. delete.
    • 如果是private或protected的,添加'_'前缀 . 如 _getName.
  • Attribute/Value 属性/变量
    • 全小写, 比如 name , firstname.
